
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docker 
```bash
$ docker-compose up -d postgres
```
## Migration

```bash
$ npx typeorm migration:create src/migration/name-migration
```

```bash
$ npm run typeorm migration:run
```
